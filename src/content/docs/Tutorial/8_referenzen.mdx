**Ressourcen und weiterführende Informationen zur Barrierefreiheit von Webinhalten**

Hier sind einige nützliche Ressourcen, Tools und Leitfäden sowie Community-Plattformen und Foren, die Ihnen helfen können, sich weiter über Barrierefreiheit zu informieren und Erfahrungen auszutauschen:

## **Ressourcen und Tools:**
1. Web Content Accessibility Guidelines (WCAG): Die offiziellen Richtlinien des World Wide Web Consortium (W3C) zur Barrierefreiheit von Webinhalten. [Link](https://www.w3.org/WAI/standards-guidelines/wcag/)
2. WebAIM: Eine Organisation, die sich auf die Förderung der Barrierefreiheit im Web spezialisiert hat. Sie bieten eine Vielzahl von Ressourcen, Tools und Schulungen an. [Link](https://webaim.org/)
3. WebAIM - Overview of Tools: Eine Übersicht über verschiedene Tools zur Barrierefreiheitsprüfung, zusammengestellt von WebAIM.
   [Übersicht der Tools](http://webaim.org/articles/tools)
4. WAVE (Web Accessibility Evaluation Tool): Ein kostenloses Online-Tool zur Überprüfung der Barrierefreiheit von Websites gemäß den WCAG. [Link](https://wave.webaim.org/)
5. Axe: Ein weiteres Tool zur automatisierten Barrierefreiheitsprüfung von Webinhalten. [Link](https://www.deque.com/axe/)
6. Color Contrast Checker: Ein Tool zur Überprüfung des Farbkontrasts zwischen Text und Hintergrund gemäß den WCAG-Richtlinien. [Link](https://webaim.org/resources/contrastchecker/)
7. VoiceOver: Ein Artikel von WebAIM, der Informationen zur Verwendung von VoiceOver, einem Bildschirmleseprogramm für Mac-Betriebssysteme, bereitstellt.
   [VoiceOver](http://webaim.org/articles/voiceover)
8. BFSG - Barrierefreiheitsgesetz [Gesetze im Internet](https://www.gesetze-im-internet.de/bfsg/BJNR297010021.html)

## **Leitfäden und Schulungen:**
1. Barrierefreie-Informationstechnik-Verordnung (BITV): Die deutschen Richtlinien zur Barrierefreiheit von Webinhalten. [Link](https://www.barrierefreie-informationstechnik.de/)
2. Accessibility Fundamentals: Ein interaktiver Kurs von Google zum Thema Barrierefreiheit im Web. [Link](https://developers.google.com/web/fundamentals/accessibility)
3. Udacity's Web Accessibility: Ein kostenloser Kurs zur Web-Barrierefreiheit. [Link](https://www.udacity.com/course/web-accessibility--ud891)

## **Community-Plattformen und Foren:**
1. Web Accessibility Slack Community: Eine Slack-Community zum Austausch von Erfahrungen und Best Practices zur Web-Barrierefreiheit. [Link](https://web-a11y.slack.com/)
2. WebAIM Community: Ein Forum von WebAIM für Diskussionen und Fragen rund um das Thema Barrierefreiheit im Web. [Link](https://webaim.org/community/)

Diese Ressourcen bieten eine Fülle von Informationen und Möglichkeiten zum Austausch mit anderen Fachleuten und Experten auf dem Gebiet der Barrierefreiheit von Webinhalten. Sie können Ihnen helfen, Ihre Kenntnisse zu erweitern und Ihre Webprojekte barrierefreier zu gestalten.
