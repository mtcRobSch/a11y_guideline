## WAVE 

WAVE (Web Accessibility Evaluation Tool) ist ein kostenloses Online-Tool von WebAIM. Es bewertet die Barrierefreiheit von Websites nach den WCAG-Richtlinien. WAVE scannt eine Webseite und findet potenzielle Hindernisse für Menschen mit Behinderungen wie fehlende Bildbeschreibungen, nicht beschriftete Formulare oder Farbkontrastprobleme. Es zeigt diese Probleme auf und gibt detaillierte Erklärungen und Lösungsvorschläge. Entwickler und Website-Betreiber können damit ihre Websites auf Barrierefreiheit prüfen und verbessern.

Gibt es als Browser-Erweiterung oder Online-Tool: 
Wave - [http://wave.webaim.org](http://wave.webaim.org/) 

 

## Weitere Tools 
Weitere Tools, die für die Barrierefreiheit von Webinhalten nützlich sind:

- Google Pagespeed: Ein Tool zur Optimierung der Ladezeit und Leistung von Webseiten. Es bietet auch einige Funktionen zur Analyse der Barrierefreiheit.
   [Google Pagespeed](https://pagespeed.web.dev/)

- Lighthouse: Ein Tool zum Test von Webseiten, der die Leistung und Qualität des Code überprüft und damit Barrierefreiheit sicherstellt.
   [Google Lighthouse](https://developers.google.com/web/tools/lighthouse/)

- General Tool (Pa11y): Ein allgemeines Tool zur Barrierefreiheitsprüfung von Webseiten, das eine Vielzahl von Tests und Analysen durchführt.
   [Pa11y](http://pa11y.org)

- Visualization Toolkit (tota11y): Ein Toolkit zur Visualisierung von Barrierefreiheitsproblemen auf Webseiten, entwickelt von Khan Academy.
   [tota11y](http://khan.github.io/tota11y)

- Color Blindness Simulator: Ein Simulator, der es ermöglicht, Webseiten aus der Perspektive von Menschen mit verschiedenen Arten von Farbenblindheit zu betrachten.
   [Color Blindness Simulator](http://www.color-blindness.com/coblis-color-blindness-simulator)

- Color Safe: Ein Tool zur Auswahl von Farbkombinationen, um sicherzustellen, dass Texte für alle Nutzer gut lesbar sind.
  [Color Safe](http://colorsafe.co)
