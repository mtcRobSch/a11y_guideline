## WAVE 

WAVE (Web Accessibility Evaluation Tool) ist ein kostenloses Online-Tool, das von der Organisation WebAIM entwickelt wurde. Es dient der Bewertung der Barrierefreiheit von Websites gemäß den Web Content Accessibility Guidelines (WCAG). WAVE scannt eine Webseite und identifiziert potenzielle Barrieren für Menschen mit Behinderungen, wie fehlende Alternative Texte für Bilder, nicht beschriftete Formulare oder Farbkontrastprobleme. Das Tool markiert diese Probleme auf der Webseite und bietet detaillierte Erklärungen und Lösungsvorschläge, um die Barrierefreiheit zu verbessern. WAVE ermöglicht Entwicklern und Website-Betreibern, ihre Websites auf Barrierefreiheit zu überprüfen und entsprechende Maßnahmen zur Verbesserung der Zugänglichkeit zu ergreifen. 

Gibt es als Browser-Erweiterung oder Online-Tool: 
Wave - [http://wave.webaim.org](http://wave.webaim.org/) 

 

## Weitere Tools 
Weitere Tools, die für die Barrierefreiheit von Webinhalten nützlich sind:

- Google Pagespeed: Ein Tool zur Optimierung der Ladezeit und Leistung von Webseiten. Es bietet auch einige Funktionen zur Analyse der Barrierefreiheit.
  Link: [Google Pagespeed](https://pagespeed.web.dev/)

- Overview of Tools: Eine Übersicht über verschiedene Tools zur Barrierefreiheitsprüfung, zusammengestellt von WebAIM.
  Link: [Übersicht der Tools](http://webaim.org/articles/tools)

- General Tool (Pa11y): Ein allgemeines Tool zur Barrierefreiheitsprüfung von Webseiten, das eine Vielzahl von Tests und Analysen durchführt.
  Link: [Pa11y](http://pa11y.org)

- Visualization Toolkit (tota11y): Ein Toolkit zur Visualisierung von Barrierefreiheitsproblemen auf Webseiten, entwickelt von Khan Academy.
  Link: [tota11y](http://khan.github.io/tota11y)

- Color Blindness Simulator: Ein Simulator, der es ermöglicht, Webseiten aus der Perspektive von Menschen mit verschiedenen Arten von Farbenblindheit zu betrachten.
  Link: [Color Blindness Simulator](http://www.color-blindness.com/coblis-color-blindness-simulator)

- Color Safe: Ein Tool zur Auswahl von barrierefreien Farbkombinationen für Webseiten, um sicherzustellen, dass Texte für alle Nutzer gut lesbar sind.
  Link: [Color Safe](http://colorsafe.co)

- VoiceOver: Ein Artikel von WebAIM, der Informationen zur Verwendung von VoiceOver, einem Bildschirmleseprogramm für Mac-Betriebssysteme, bereitstellt.
  Link: [VoiceOver](http://webaim.org/articles/voiceover)
